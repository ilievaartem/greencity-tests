# Тест-кейси для сторінки Events (GreenCity)

## Test-1: Перевірка переходу на сторінку детальної інформації про подію (Позитивний)
Відкрийте сторінку [GreenCity Events](https://www.greencity.cx.ua/#/greenCity/events).

1 Натиснути лівою кнопкою миші на кнопку "Більше" картки будь-якої події - Відбувається перехід на сторінку з повним описом обраної події
2 Перевірити наявність кнопки "Back to events" - Кнопка відображається у верхній частині сторінки 

## Test-2: Пошук події за точною назвою (Позитивний)
Знайдіть подію з існуючою назвою (наприклад, "123123").
Відкрийте сторінку "Події".

1 Ввести назву події у поле пошуку "Search" після натиску на кнопку лупи -  У списку подій відображається лише подія з цією назвою 
2 Очистити поле пошуку, натиснувши на "Х" у полі - Поле пошуку пусте, відображається повний список усіх подій |

## Test-03: Пошук події за неіснуючим запитом (Негативний)
Відкрийте сторінку "Події".

1 Ввести у поле пошуку набір випадкових символів - "@#$!123" - Список подій зникає та виводитьс 
2 Перевірити наявність повідомлення про відсутність результатів - Відображається напис "Ми не знайшли жодних результатів, що відповідають цьому запиту"
